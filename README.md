## 🚀 Featured projects

### [Slice](https://sliceapp-pi.vercel.app) — Expense splitting app
> Split expenses with friends. No awkward conversations.

Full-stack expense tracker built for real-world use. Groups, splits, settlements, receipt uploads, email notifications, and a greedy debt-minimization algorithm that reduces N people's debts to at most N-1 transactions.

`MongoDB` `Express.js` `React.js` `Node.js` `Cloudinary` `Nodemailer` `node-cron` `JWT` `Vercel` `Render`

### [Car Rental Platform](https://car-rental-website-olive-gamma.vercel.app) — Full stack MERN
> Role-based car rental system for owners and users.

Full-stack rental platform with Owner and User dashboards, vehicle search and filtering, booking workflows, and JWT-based auth with custom Express middleware.

`MongoDB` `Express.js` `React.js` `Node.js` `JWT` `REST API`
